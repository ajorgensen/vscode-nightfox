# Installation

1. Open the **Extensions** sidebar in VS Code / `View â†’ Extensions`
2. Search for "Nightfox" by **Keifer Erikson**
3. Click **Install** 
4. Navigate to File > Preferences > Color Theme > **Nightfox**
